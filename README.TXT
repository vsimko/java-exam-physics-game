
Import this archive as a project to Eclipse

The project contains two lauch configurations located in the ".launch" directory.
- PhysicsGame (Client).launch : Server is implemented in javatest.game.Server
- PhysicsGame (Server).launch : Client is implemented in javatest.game.client.Client

- Animated images are located in "src/images"
- Source files of animated images are located in "imgdev" use GIMP to edit
- lib/commons-io-2.0.1.jar is used for reading images from within the JAR file
